# digital-sundae

Explorations in testing SPA web apps with React.js, Node.js, Jest, and React Testing Library.

---

## Quick Start

_Client-side:_
```sh
cd client
npm install
npm run start
# in browser, visit localhost:3000
```

_Server-side:_
```sh
cd server
npm install
npm run start
```
